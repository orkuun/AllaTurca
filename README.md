Türk müziğinde kullanılan ritm kalıplarını vuruş desenine göre düzenleyip, 
partisyona vurmalı çalgı partisi olarak ekleyen bir eklenti.

A plugin that organizes the rhythmic patterns used in Turkish music 
according to their beat patterns and adds them to the score as a percussion instrument part.
